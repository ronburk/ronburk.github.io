Stumbling Around V8

V8 is Google's JavaScript engine. It is notably used inside of Google's Chrome web browser, Node.js, and Electron.

As an open-source C++ library, V8 could theoretically be used by any C++ programming project that wants to embed a JavaScript interpreter. In practice, however, using V8 is pretty non-trivial.
